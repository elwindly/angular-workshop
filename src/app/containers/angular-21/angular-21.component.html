Angular 21 highlights 2025 — v21.0.0

  • Zoneless by default: new apps no longer include Zone.js; migration support and provideZonelessChangeDetection()
  • Vitest as the new default, stable test runner ng g @schematics/angular:refactor-jasmine-vitest
  • Signal-based forms: experimental signal forms and new debounce() rule
  • Angular Aria (@angular/aria): developer preview — headless, accessible UI patterns (Accordion, Combobox, Grid, Listbox, Menu, Tabs, Toolbar, Tree); signal-based and fully stylable Official guide
  • MCP server stable: seven tools for AI agents (best practices, search_documentation, find_examples, onpush_zoneless_migration, modernize, ai_tutor, list_projects)
  • Zoneless by default migration support and improvements for server bootstrapping (BootstrapContext)
  • Control flow and template improvements: better typing, regular expressions in template expressions (e.g. with @let), improved @for reactivity
  • Enhanced compiler checks: host binding type-checking enabled by default and new diagnostics
  • Forms UX improvements: FormArrayDirective, better interop between signal forms and reactive forms
  • Router: lastSuccessfulNavigation as a signal; various router memory and fix improvements
  • Tooling/migrations: new migrations for control flow, ngClass/ngStyle to class/style bindings, standalone support, RouterTestingModule migration
  • Template and API polish: SimpleChanges now generic; KeyValue pipe supports objects with optional keys; customizable @defer viewport options (e.g. rootMargin for on viewport)
  • Enter/leave animations: new animate.enter and animate.leave APIs for DOM insertion and removal Animations guide
  • DevTools: built-in Signals formatter (enable custom formatters in Chrome/Firefox); Transfer State tab for SSR inspection
  • CDK overlays can use the browser’s native Popover API for improved accessibility
  • angular.dev/ai: dedicated docs, prompts, and rules for building AI-powered applications with Angular
  • Compatibility: minimum TypeScript 5.9 (no older); Node ^20.19.0 || ^22.12.0 || ^24.0.0
  • Tailwind CSS integration: improved support and official guidelines for using Tailwind CSS with Angular Official guide
  • Theming improvements: new CSS variables, Material system token utility classes, and better integration with Material Official guide
