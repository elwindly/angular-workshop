Angular 21 highlights 2025 — v21.0.0

  • Navigation API experimental support (platform integration)
  • Signal-based forms: experimental signal forms and new debounce() rule
  • Zoneless by default migration support and improvements for server bootstrapping (BootstrapContext)
  • BootstrapContext for improved server bootstrapping (server-side API changes)
  • Control flow and template improvements: better typing and template expression fixes (regex in templates, improved @for reactivity)
  • Enhanced compiler checks: host binding type-checking enabled by default and new diagnostics
  • Forms UX improvements: FormArrayDirective, better interop between signal forms and reactive forms
  • Router: convert lastSuccessfulNavigation to a signal; various router memory/fix improvements
  • Tooling/migrations: new migrations for control flow, ngClass/ngStyle conversions and standalone support
  • Platform and runtime: move toward drop of Zone defaults, remove deprecated metadata and APIs (moduleId, ApplicationConfig export refactor)
  • Compatibility notes: minimum TypeScript raised (no older than 5.9) and Node.js runtime requirements updated
  • Release notes: https://github.com/angular/angular/releases/tag/21.0.0

This container is a concise reference — for full details see the official release notes linked above.
