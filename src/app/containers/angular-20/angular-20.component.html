Angular 20 new features 2025 MAY 26

  • Stabilized Signal-based APIs: toSignal(), toObservable(), and linkedSignal() now stable and production-ready
  • Control Flow Syntax: Stabilized @'if', @'for', @'switch', @'case', and @'let' for cleaner templates (ngIf/ngFor/ngSwitch deprecated)
  • Zoneless Change Detection: Moved to developer preview with improved performance without Zone.js 20.2 production ready
  • Experimental MCP server to help with LLM codegen
  • Template Expression Improvements: Support for exponentiation operator (**), in keyword, void operator, and tagged template literals in expressions
  • CDK Drag&Drop improvement to allow copying items between lists
  • Dynamic Component Features: Add input binding support, two-way bindings, and directives on dynamically-created components
  • Router Enhancements: Asynchronous redirects support, ability to directly abort navigation, improved type safety on route guards
  • Forms Improvements: markAllAsDirty method added, ability to reset forms without emitting events
  • Route visualization and signal graph in Angular DevTools
  • Animations: New animation APIs for more dynamic and flexible animations Link to the official page
  • TypeScript 5.8+ Requirement: Dropped support for TypeScript older than 5.8
  • Node.js Updates: Minimum Node.js 20.19, with support for 22.12 and 24.0+ (dropped support for Node 18 and 22.0-22.10)
